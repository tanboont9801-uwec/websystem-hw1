My physics teacher said that the best way to learn physics is to double-major in computer science.
 Then she gave us the following homework problem:

  Write a Javascript class that models a projectile. Provide a constructor that accepts the
  projectile's initial velocities on both the x- and y-axes. Provide also a method for
  getting the projectile's position at time t.

 I will not take a programming course until my 5th grade year, but I watched all of last
 summer and managed to piece together the following solution:

 class Projectile {
   constructor(velocityX, velocityY) {
      this.velocityX = velocityX;
      this.velocityY = velocityY;

   }
  getPosition(t) {
     return [this.velocityX * t, -4.9 * t * t + this.velocityY * t];

   }
 }
